# R26-IT-016
# Intelligent Multilingual Mobile Learning System for Early Childhood Education in Sri Lanka

## Project ID
R26-IT-016

## Project Overview
This project focuses on developing an AI-powered multilingual mobile learning system designed for Nursery and Grade 1 children in Sri Lanka. The system aims to improve early childhood education by integrating interactive learning, handwriting recognition, cultural education, storytelling, and adaptive learning techniques.

The platform supports Sinhala, Tamil, and English languages and provides engaging educational activities that help children improve literacy, numeracy, cultural awareness, and moral development.

---

## Research Problem
Most existing educational mobile applications mainly support English-based learning and lack culturally relevant content for Sri Lankan children. Current systems also provide limited handwriting evaluation, lack real-time corrective feedback, and do not support adaptive personalized learning.

Additionally, topics such as Sri Lankan history, provinces, biodiversity, religions, and moral education are rarely integrated into modern digital learning platforms for early childhood education.

---

## Proposed Solution
The proposed solution is an Intelligent Multilingual Mobile Learning System that integrates:

- AI-based handwriting recognition
- Stroke-level writing analysis
- Interactive storytelling
- Gamification techniques
- Learning analytics
- Adaptive personalized learning
- Cultural and historical education modules

The system provides real-time feedback, multilingual learning support, and progress tracking for teachers and parents.

## Additional Functionalities

### AI-Powered Handwriting Evaluation
The system captures stroke-level handwriting data including:
- Stroke order
- Stroke direction
- Writing speed
- Shape accuracy
- Writing consistency

Machine learning algorithms analyze handwriting behavior and provide immediate corrective feedback using animations, voice guidance, and visual indicators.

## Installation Guide

### Prerequisites
- Flutter SDK
- Android Studio
- Firebase Account
- Python 3.x
- TensorFlow Lite

### Setup Instructions

1. Clone the repository

```bash
git clone https://github.com/AkashMalinga/R26-IT-016.git
```

2. Navigate to project directory

```bash
cd R26-IT-016
```

3. Install dependencies

```bash
flutter pub get
```

4. Run the application

```bash
flutter run
```
